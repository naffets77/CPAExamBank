Creating a New Class is Easy!

Enter the following information:

  1. Language: Provide the language of study
  2. Class Name: Choose a descriptive name to identify your class.
    **Hint: Use a standardized name to help keep your classes organized.

    Ex. Spanish Fall 2012 Period 1 = SP_FALL2012_P1

    Ex. French Quarter 1 2011 9:10am - 10:00am = FRQ1_2011_910A

  3. Expected Class Size: Entering the expected class size will allow you to keep track of the total number of students registered.

**Tokens: After a class is created you will receive a token that must be distributed to your students. This token will allow your students to register into the class that has just been created.

Select Class

Please select the class you wish to manage from the drop down menu.

The classes tab will allow you to:
  • Create a class
  • View student details
  • Send class wide messages
  • Adjust curriculum settings
 

Create a New Class:

Use the 'New Class' button located in the upper right corner to create your class. You will need to enter the course language, a class name and expected class size.

**Note: A class is defined to be a course that is one semester or one quarter's time frame.

What is a topic?

Topics are a way to organize how you want your students to be able to communicate.

How should I organize my Topics?

Topics can be organized in many different ways. You may want to organize them based on the content you are currently covering (Food, Greetings, Sports), by the week of the class, or possibly representing a chapter in a book.

Assigning Topics to classes

Topics are assigned to a class. You can do this in the class tab after selecting a particular class. You can then choose which topics are available for a given class at any time.

Topic Activities

After you create a topic, you can create topic activies. The activities are what a student will use to practice his language skills.

Lesson Plan :

Select the lesson plan you wish to manage. You can use a default lesson plan, or use a default lesson plan as a starting point for your own custom lesson plan.

Description
Provide some context about what will be talked about.
Prompts Prompts are how a studen will interact with other students. They are optional, as you may want to set up a scence with a description and allow students to speak about that. However, if you want to provide a more focused practice, you can provide a prompt which the student can use to provide a starting point when practicing the language.

The prompt has two parts, the prompt and the hint. Depending on the level of instruction, you may write the prompt in your native language, or in the practice language. Both students will be able to view the prompt, however only the listening student will be able to view the hint. For example if you provide the prompt in the naitive language, and the hint in the forgien language, the student speaking can be tasked with translating the prompt while the listening student can provide assistance.
